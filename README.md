## 🚨 The Challenge

Finding the exact blood type during medical emergencies is often chaotic, stressful, and time-consuming. Patients' families usually rely on forwarding social media messages or calling blood banks manually. **This critical delay can cost lives.**

---

## 💡 The Solution

**RaktSetu** automates the emergency SOS process. When an urgent blood request is raised, the platform instantly filters nearby eligible donors (enforcing a strict 90-day health cooldown) and alerts them in real time.

## ✨ Core Features

| Feature | Description |
| :--- | :--- |
| 🎯 **Smart Matching** | Instantly connects donors and patients based on exact blood group and location. |
| ⏳ **Eligibility Engine** | Automatically tracks safe donation intervals (90-day cooldown) to protect donor health. |
| 🚨 **Live Emergency Board** | A real-time, auto-refreshing dashboard for urgent blood requests. |
| 👥 **Multi-User Portals** | Dedicated, secure dashboards tailored for **Donors**, **Hospitals/Blood Banks**, and **Administrators**. |

---

## 🛠️ Tech Stack & Architecture

Built with a focus on robust web engineering principles, security, and raw performance over heavy frameworks.

- **Frontend:** HTML5, CSS3, Vanilla JavaScript, Responsive Design
- **Real-Time Logic:** Asynchronous AJAX polling for instant dashboard updates without page reloads
- **Backend:** PHP 8.0+
- **Database:** MySQL with a strictly Normalized Schema
- **Data Access:** PDO (PHP Data Objects)
- **Security:** 
  - Secure `bcrypt` password hashing
  - Prepared statements to prevent SQL injections
  - Role-Based Access Control (RBAC) for admins and users
