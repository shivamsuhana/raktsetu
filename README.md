# 🩸 RaktSetu — Emergency Blood Donor Network

RaktSetu is a real-time emergency blood-matching platform designed to instantly connect blood donors, patients, and hospitals based on proximity and blood group.

[👉 View Live Demo](http://raktsetu.page.gd/)

---

## 🚨 The Challenge
Finding the exact blood type during medical emergencies is often chaotic and stressful. Patients' families usually rely on social media forwards or calling blood banks manually. This critical delay can cost lives.

## 💡 The Solution
RaktSetu automates the emergency SOS process. When a request is raised, the platform instantly filters nearby eligible donors (enforcing a strict 90-day health cooldown) and alerts them in real time. 

### Core Features

*   **Smart Matching:** Instantly connects donors and patients based on blood group and location.
*   **Eligibility Engine:** Automatically tracks safe donation intervals.
*   **Live Emergency Board:** Real-time dashboard for urgent blood requests.
*   **Multi-User Roles:** Dedicated portals for Donors, Hospitals, and Administrators.

---

## 🛠 Tech Stack

Built from scratch, emphasizing core web engineering principles over heavy frameworks.

*   **Frontend:** HTML5, CSS3, Vanilla JavaScript
*   **Logic:** Real-time AJAX polling for instant dashboard updates
*   **Backend:** PHP 8, PDO 
*   **Database:** MySQL (Normalized Schema)
*   **Security:** Secure password hashing, Prepared statements to prevent SQL injections
